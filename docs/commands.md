# 명령어

## 명령어의 구조

명령어는 다음과 같은 구조를 가진다.

```
# [명령어]
> 봐
```

```
# [대상] [명령어]
> 홍심이 봐
> 도토리 주워
```

```
# [대상] [꾸미는말] [명령어]
> 수지 "힘껏" 뽀뽀
=> 당신은 수지에게 힘껏 뽀뽀합니다.
```

```
# [내용] [명령어]
> 안녕쌉싸리와요 말
```

```
# [대상] [내용] [명령어]
> 가방 생수 꺼내
> 상인 담배 사
> 수지 지금 시간있어? 귓속말
```

```
# [대상] [내용] [꾸미는말] [명령어]
> 가방 숟가락 "비장하게" 꺼내
=> 당신은 가방에서 숟가락을 비장하게 꺼냅니다.
```
